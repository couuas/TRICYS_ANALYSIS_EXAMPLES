# AI模型分析结果

### 深度解读分析报告

---

#### 1. 全局敏感性分析（基于性能指标总表）

**总体趋势量化描述：**
- **Startup_Inventory**：随Fueling_Efficiency提升呈指数衰减趋势。从0.01→0.9时，启动库存从202.47kg降至2.69kg（降幅98.7%），其中0.01→0.1阶段降幅最陡峭（95.1%）。
- **Doubling_Time**：仅当Fueling_Efficiency≥0.4时存在有效值，且随效率提升线性递减（486.87→357.06天，降幅26.7%）。效率0.4→0.5时降幅最大（-51.38天）。
- **Self_Sufficiency_Time**：类似地，效率≥0.1时出现有效值，从13.35天（0.1）降至7.37天（≥0.7），降幅44.8%。效率0.1→0.2阶段降幅最显著（-29.8%）。
- **Required_TBR**：效率≤0.05时需无限增殖（inf），0.05→0.2阶段从1.1719骤降至1.0469（降幅10.7%），之后平缓收敛至1.0156（≥0.7）。

**敏感性排序：**
- **最敏感指标**：Startup_Inventory（效率0.01→0.1时库存变化率达95.1%），其次是Required_TBR（效率0.05→0.2时TBR降幅10.7%）。
- **最不敏感指标**：Self_Sufficiency_Time（效率≥0.7后完全不变），Doubling_Time在效率≥0.5后变化率<5%/0.1增量。

---

#### 2. 动态过程分析（基于关键数据切片）

**初始阶段 vs 结束阶段行为差异：**
- **初始阶段（t<20h）**：所有效率下的库存均从3500g开始快速消耗，但低效率（0.01）在0.5h时已降至-5672.84g（严重透支），而高效率（0.9）仅降至3499.98g（几乎无损）。
- **结束阶段（t≈20000h）**：高效率组（≥0.4）库存均达11874g以上，形成稳定增殖；低效率组（≤0.2）仍为负值（-4338.63g~6469.95g），系统崩溃。

**转折点阶段物理过程（以0.4效率为例）：**
- **t≈182h时**库存达最小值-314.508g，随后转为正增长（89.8186g→698.494g）。这表明：
  - 此时氚燃烧速率与增殖速率达到动态平衡；
  - 之后TBR>1的优势开始显现，库存进入净增长阶段；
  - 高效率组（≥0.5）更早出现转折（如0.9组在t≈181h时已812.085g）。

---

#### 3. 综合结论与建议

**运行场景影响与权衡：**
- **高效率（≥0.5）优势**：显著降低启动库存（<3.5kg）、缩短自持时间（7.37天）、减少对TBR的苛求（1.0156），但需付出更高等离子体约束技术成本。
- **低效率（≤0.2）风险**：需极高TBR（≥1.0469）或无限库存补给，实际中可能因库存耗尽导致系统停堆（如0.01效率库存终值-198895g）。

**优化建议：**
1. **阈值选择**：Fueling_Efficiency应至少>0.4（临界点），否则需TBR>1.0273，可能超出包层设计能力；
2. **经济性权衡**：效率从0.7提升至0.9时，性能增益有限（Self_Sufficiency_Time不变，Doubling_Time仅降7.2%），可优先考虑0.6~0.7区间；
3. **启动策略**：低效率时需预存超200kg氚（不现实），建议通过阶段性提升效率（如从0.4→0.6）逐步降低库存需求。

**设计启示**：  
燃料循环系统需与等离子体约束性能协同设计。若约束技术难以突破0.5效率，则必须开发TBR>1.0234的高效增殖包层；反之可放宽对包层的要求。
