#### **1. 全局敏感性分析：性能指标的总体趋势与敏感度量化**

基于**性能指标总表**的数据，对 `plasma.Fueling_Efficiency` 的变化所引发的系统响应进行全局分析：

##### **（1）总体趋势量化描述**

- **启动库存（Startup Inventory）**：
  - 随着 `plasma.Fueling_Efficiency` 从 0.01 增加到 0.9，**启动库存从 202.47 kg 单调递减至 2.69 kg**。
  - 在效率低于 0.1 时，下降速率极快：从 0.01 到 0.1，库存由 202.47 降至 9.94，降幅达 **95.1%**。
  - 当效率 ≥ 0.4 后，下降趋缓：从 0.4 到 0.9，库存仅减少约 38.7%（3.82 → 2.69），表明高效率下进一步优化空间有限。

- **自持时间（Self Sufficiency Time）**：
  - 该指标在 `plasma.Fueling_Efficiency = 0.1` 时首次出现有效值（13.35 天），随后持续下降。
  - 从 0.1 到 0.8，自持时间从 13.35 天降至 7.37 天，**累计下降 44.7%**。
  - 从 0.8 到 0.9，自持时间**保持稳定在 7.37 天**，无任何变化，说明达到“饱和”状态。

- **翻倍时间（Doubling Time）**：
  - 随 `plasma.Fueling_Efficiency` 提升而单调递减。
  - 从 0.01 到 0.9，翻倍时间从 `nan`（未收敛）降至 357.06 天，**下降幅度超过 80%**。
  - 在效率 ≥ 0.4 后，下降速率放缓：0.4→0.5 时减少 51.38 天，0.5→0.6 减少 31.18 天，0.6→0.7 减少 20.94 天，0.7→0.8 减少 15.00 天，0.8→0.9 减少 11.31 天，呈现**显著的边际递减效应**。

- **所需 TBR（Required TBR）**：
  - 在 `plasma.Fueling_Efficiency = 0.01` 时为 `inf`，表示无法满足约束条件（即无法实现可持续运行）。
  - 从 0.05 开始，所需 TBR 从 1.1719 逐步下降至 1.0156，且在 0.7~0.9 范围内**稳定于 1.0156**。
  - 从 0.05 到 0.7，所需 TBR 下降了约 **13.0%**；而从 0.7 到 0.9，**无任何变化**，表明系统已达到可接受的最小所需再生率阈值。

##### **（2）敏感度排序：最敏感与最不敏感指标**

| 指标 | 敏感度等级 | 理由 |

#### **2. 动态过程分析：关键阶段行为揭示的物理机制**

基于“关键动态数据切片”中的三阶段数据，深入剖析系统演化过程。

##### **（1）初始阶段（前 20 个数据点，间隔 2 小时）**

- **所有曲线均起始于相同初始库存（3500 g）**，但随时间推移，**差异迅速扩大**。
- **低效率组（C1–C4）**：库存急剧下降，甚至在 0.5 小时内进入负值（如 C1 在 0.5 小时降至 -5672.84 g），表明**燃料消耗远超再生能力**，系统处于严重失衡状态。
- **高效率组（C5–C10）**：库存下降缓慢，在 10 小时内仍维持正值（如 C10 在 10.35 小时为 2200.21 g），并开始回升趋势。
- **关键观察**：**效率越低，初始阶段的库存衰减越快，且提前进入负增长区域**，这直接导致后续需更大初始库存来弥补。

> **物理意义**：初始阶段是系统“燃烧期”，若燃料注入效率低下，则库存迅速耗尽，形成“负向积累”，迫使设计者投入更多初始氚以维持运行。

##### **（2）转折点阶段（围绕 `sds.inventory&plasma.Fueling_Efficiency=0.4` 最小值）**

- 在时间约 **182~195 小时** 之间，`C5`（eff=0.4）的库存达到最低点 **-314.508 g**，此后开始回升。
- 对比其他组：
  - **低效率组（C1–C4）**：库存持续下降，且最低点出现在更早时间（如 C1 在 181.35 小时为 -78953.6），之后虽略有波动，但始终未见明显反转。
  - **高效率组（C6–C10）**：库存从负值逐渐回升，并在 185 小时后持续正增长。
- **关键发现**：**`plasma.Fueling_Efficiency = 0.4` 是系统从“净消耗”转向“净生产”的临界点**。其库存曲线在 182 小时左右出现**局部最小值**，之后开始上升，标志着系统首次实现**氚的净生成**。

> **物理意义**：该阶段明确揭示了**系统存在一个“临界注入效率”**。当 `plasma.Fueling_Efficiency ≥ 0.4` 时，再生速率首次超过消耗速率，系统进入自我维持状态。此临界点与性能指标中“自持时间首次出现”的数值（7.87 天 ≈ 189 小时）高度吻合，验证了其一致性。

##### **（3）结束阶段（后 20 个数据点，时间接近 20000 小时）**

- 所有组别的库存最终均转为正值并持续上升。
- **高效率组（C9, C10）**：在 20000 小时处库存分别达到 **14908.7 g** 与 **14908.7 g**，而低效率组（C1）仅为 **-197418 g**（仍为负值）。
- **特别注意**：尽管 `C1` 和 `C2` 在末期仍为负值，但其数值绝对值增长缓慢，表明系统已进入“渐进衰竭”而非“加速崩溃”。
- **对比可见**：**效率越高，最终库存增长越快、总量越大**，体现长期运行下的优势累积效应。

> **物理意义**：结束阶段反映了系统的长期稳定性。只有在足够高的注入效率下，系统才能完成从“依赖外部输入”到“自主供能”的转变，并实现氚库存的指数级增长。

#### **3. 综合结论与优化建议**

结合全局趋势与动态过程分析，得出以下综合结论：

##### **核心发现总结**

1. **`plasma.Fueling_Efficiency` 对系统性能具有决定性影响**，尤其在**启动阶段和临界点附近**。
2. **存在一个关键效率阈值（≈0.4）**：当效率 ≥ 0.4 时，系统首次实现自持（自持时间首次出现），并开启净增殖路径。
3. **系统存在明显的“边际效益递减”现象**：
   - 在效率 < 0.4 区间，提升效率带来的收益巨大（尤其是启动库存与翻倍时间）；
   - 在效率 ≥ 0.7 区间，所有性能指标（自持时间、所需 TBR、翻倍时间）趋于饱和，进一步提升几乎无收益。
4. **启动库存是最大瓶颈**：其对效率极度敏感，是设计初期最大的经济与工程挑战。

##### **利弊权衡分析**

| 运行场景 | 优点 | 缺点 |
|--------|------|------|
| **低效率（≤0.1）** | 技术门槛低，易于实现 | 启动库存极高（>200 kg），翻倍时间不可行，所需 TBR 无法满足，系统无法自持 |
| **中等效率（0.2~0.4）** | 自持时间短，启动库存可控 | 仍需较高初始投入，翻倍时间较长 |
| **高效率（≥0.7）** | 自持时间稳定在 7.37 天，翻倍时间最短（357 天），所需 TBR 最优（1.0156） | 提升效率的边际收益微弱，可能面临材料/工程极限 |

##### **系统设计与运行优化建议**

1. **优先保障 `plasma.Fueling_Efficiency` ≥ 0.4**：
   - 这是系统能否实现自持的根本前提，应作为设计底线。
   - 建议将 0.4 作为工程目标，避免系统陷入“永远无法自持”的死循环。

2. **聚焦效率 0.4~0.7 区间的技术攻关**：
   - 此区间是性能跃迁的核心区，每提升 0.1 效率，可带来显著的启动库存降低（3.82 → 2.95 → 2.80 → 2.69）和翻倍时间缩短（486.87 → 383.37 天）。
   - 建议投入资源优化燃料注入系统（如中子源、靶材、回收率），以突破 0.7 门槛。

3. **避免过度追求 >0.7 的效率**：
   - 超过 0.7 后，所有关键指标趋于饱和，投入产出比极低。
   - 若现有技术难以突破 0.7，应接受当前最优解，而非盲目追求理论极限。

4. **建立“效率-库存”耦合设计框架**：
   - 基于 `plasma.Fueling_Efficiency` 与 `Startup_Inventory` 的强非线性关系，建议采用反向设计法：设定可接受的初始库存（如 ≤5 kg），倒推所需最低效率（约为 0.5~0.6）。

5. **强调“临界效率”概念的工程应用**：
   - 在系统调试阶段，应重点监测 `sds.inventory` 是否在约 180~190 小时出现拐点，用以验证系统是否真正进入自持状态。

---

### **最终结论**

`plasma.Fueling_Efficiency` 是聚变堆氚燃料循环系统性能的**核心控制变量**。其影响呈“非线性跳跃式”特征：**0.4 是系统生死线，0.7 是性能天花板**。在工程实践中，应以 **0.4 为安全边界，0.7 为优化上限**，在 0.4~0.7 区间内寻求效率与成本的最佳平衡。任何设计或运行策略都必须以此为核心依据，方能实现高效、可靠、可持续的氚燃料循环。
